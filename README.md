# Wordle With Friends
### Social Wordle


Wordle with friends is a website created for my friend group to track each others scores.

### Current Features
- Upload scores
- Stats (streak, worse word, etc)
- Compare scores 1v1 


This is a Next.JS, supabase, and tailwindCSS project.
